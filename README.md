# versioned

Quickly increment major/minor/patch version in VERSION file.

## Getting Started

Display current version:

```bash
versioned
```

Update minor version in `VERSION` file:

```bash
versioned -major
```

Update minor version in `VERSION` file:

```bash
versioned -minor
```

Update patch version in `VERSION` file:

```bash
versioned -patch
```
